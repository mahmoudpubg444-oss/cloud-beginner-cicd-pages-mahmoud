What is CI/CD?

CI/CD stands for Continuous Integration and Continuous Deployment. It is a process used in modern cloud systems to automate building, testing, and deploying applications. Continuous Integration means developers frequently push code changes to a shared repository where automatic checks run. Continuous Deployment means the application is automatically updated after successful tests. This helps developers deliver software faster and with fewer errors.

My Pipeline

When I push code to the main branch on GitHub, a GitHub Actions workflow starts automatically. The workflow checks that all required files exist. If everything is correct, the website files are uploaded and deployed to GitHub Pages. After deployment, the live website updates automatically.
